Network Security Cryptography Infrastructures

ChainOfProduct Secure Document Platform

Built a multi-service secure document platform for controlled disclosure of transaction data across organizations. The system combines a FastAPI application, a Java cryptographic library, PostgreSQL, a certificate authority service, and a group-management server in a segmented virtualized infrastructure.

Group project developed in a team of 3. Although having an overall understanding of the project, My contribution focused on infrastructure design and deployment, frontend integration, database management, certificate authority modelling, firewall and secure connections configuration.

Highlights

  • Implemented document protection with AES-GCM, RSA-OAEP, and RSASSA-PSS.
  • Designed auditable sharing workflows with certificate-based identity validation.
  • Secured communication channels with TLS and isolated services across multiple virtual machines.

Tech stack

  • Python / FastAPI
  • Java / Maven
  • PostgreSQL
  • Nginx, TLS, PKI, VirtualBox

Software Security Python Static Analysis

Static Analyzer for Python Web Vulnerabilities

Developed a Python-based static analysis tool that inspects program slices and flags insecure data flows from untrusted sources to sensitive sinks. The analyzer models explicit and implicit flows, tracks sanitization, and produces structured JSON reports for vulnerability findings.

Highlights

  • Implemented analysis over Python ASTs to reason about sources, sinks, and sanitizers.
  • Supported multiple vulnerability patterns including XSS, SQL injection, SSRF, and path traversal.
  • Handled explicit and implicit information flows with deterministic output formatting.

Tech stack

  • Python
  • AST parsing
  • JSON-based vulnerability patterns
  • Static program analysis
