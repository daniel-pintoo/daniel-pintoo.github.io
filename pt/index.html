Segurança em Redes e Sistemas Criptografia Infraestruturas

ChainOfProduct Plataforma de Documentos Seguros

Plataforma segura multi-serviços para a divulgação controlada de dados de transações entre organizações. O sistema combina uma aplicação FastAPI, uma biblioteca criptográfica em Java, PostgreSQL, um serviço de autoridade certificadora e um servidor de gestão de grupos numa infraestrutura segmentada e virtualizada.

Projeto de grupo implementado com uma equipa de 3 estudantes. Apesar de ter uma compreensão global do projeto, a minha contribuição prevaleceu no desenho e lançamento da infraestrutura, integração do frontend, gestão da base de dados, modelação da autoridade certificadora, configuração das firewalls e ligações seguras.

Destaques

  • Implementámos proteção de documentos com AES-GCM, RSA-OAEP e RSASSA-PSS.
  • Desenhámos fluxos auditáveis de partilha com validação de identidade baseada em certificados.
  • Protegemos canais de comunicação com TLS e isolámos serviços em várias máquinas virtuais.

Tecnologias

  • Python / FastAPI
  • Java / Maven
  • PostgreSQL
  • Nginx, TLS, PKI, VirtualBox

Cibersegurança Forense Análise de Redes Investigação de Incidentes

Investigação MEMELgate de Cibersegurança Forense

Realizei um projeto universitário de cibersegurança forense em três etapas focado na investigação de uma campanha de phishing, recuperação de artefactos escondidos, análise de imagens forenses de discos, reconstrução e exfiltração de dados através da análise de tráfego de rede.

Projeto universitário realizado com uma equipa de 3 estudantes desenvolvido ao longo de três laboratórios progressivos, com relatórios forenses, reconstrução de timelines e justificações técnicas baseada em provas.

Destaques

  • Recuperámos artefactos suspeitos e ocultos a partir de ficheiros, websites e outras evidencias digitais relacionadas.
  • Analisámos imagens de discos para determinar a proveniência de ficheiros suspeitos, atividade de potenciais atacantes e comportamentos anti-forenses.
  • Investigámos capturas de rede e tráfego HTTPS desencriptado para reconstruir eventos de exfiltração e hipóteses de atribuição de responsabilidade.

Tecnologias

  • Kali Linux
  • Wireshark
  • FTK Imager
  • Análise forense de discos, sistemas de ficheiros e redes

Segurança em Software Python Análise Estática

Static Analyzer for Python Web Vulnerabilities

Desenvolvi uma ferramenta de análise estática em Python que analisa slices de programas e sinaliza fluxos de dados inseguros vindos fontes não confiáveis até sinks sensíveis. O analisador define fluxos explícitos e implícitos, acompanha sanitizações e produz relatórios em formatoJSON estruturados para as vulnerabilidades encontradas.

Destaques

  • Implementámos a análise estática sobre ASTs de Python para analisar fontes, sinks e sanitizadores.
  • Definimos vários padrões de vulnerabilidade, incluindo XSS, SQL injection, SSRF e path traversal.
  • Tratámos fluxos de informação explícitos e implícitos com a formatação determinística dos resultados.

Tecnologias

  • Python
  • Parsing de ASTs
  • Padrões de vulnerabilidades em JSON
  • Análise estática de código python
