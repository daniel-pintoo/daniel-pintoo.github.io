Seguranca de Backend Criptografia Aplicada Infraestrutura

ChainOfProduct Secure Document Platform

Desenvolvi uma plataforma segura multi-servico para divulgacao controlada de dados de transacoes entre organizacoes. O sistema combina uma aplicacao FastAPI, uma biblioteca criptografica em Java, PostgreSQL, um servico de autoridade certificadora e um servidor de gestao de grupos numa infraestrutura segmentada e virtualizada.

Projeto de grupo desenvolvido numa equipa de 3. Apesar de ter uma compreensao global do projeto, a minha contribuicao incidiu no desenho e deployment da infraestrutura, integracao do frontend, gestao da base de dados, modelacao da autoridade certificadora, configuracao de firewalls e ligacoes seguras.

Destaques

  • Implementei protecao de documentos com AES-GCM, RSA-OAEP e RSASSA-PSS.
  • Desenhei fluxos auditaveis de partilha com validacao de identidade baseada em certificados.
  • Protegi canais de comunicacao com TLS e isolei servicos em varias maquinas virtuais.

Tecnologias

  • Python / FastAPI
  • Java / Maven
  • PostgreSQL
  • Nginx, TLS, PKI, VirtualBox

Forense Digital Analise de Rede Investigacao de Incidentes

MEMELgate Digital Forensics Investigation

Realizei um projeto universitario de forense digital em tres etapas focado na investigacao de uma campanha de phishing, recuperacao de artefactos ocultos, analise de imagens forenses de disco e reconstrucao de exfiltracao de dados atraves de analise de trafego de rede.

Projeto universitario de equipa desenvolvido ao longo de tres laboratorios progressivos, com relatorios forenses, reconstrucao de timelines e justificacao tecnica baseada em evidencia.

Destaques

  • Recuperei artefactos suspeitos e ocultos a partir de ficheiros, websites e outra evidencia digital relacionada.
  • Analisei imagens forenses de disco para rastrear proveniencia de ficheiros, atividade de atacantes e comportamento anti-forense.
  • Investiguei capturas de rede e trafego HTTPS desencriptado para reconstruir eventos de exfiltracao e hipoteses de atribuicao.

Tecnologias

  • Kali Linux
  • Wireshark
  • FTK Imager
  • Forense de disco, sistema de ficheiros e rede

Seguranca de Software Python Analise Estatica

Static Analyzer for Python Web Vulnerabilities

Desenvolvi uma ferramenta de analise estatica em Python que inspeciona fatias de programas e sinaliza fluxos de dados inseguros desde fontes nao confiaveis ate sinks sensiveis. O analisador modela fluxos explicitos e implicitos, acompanha sanitizacao e produz relatorios JSON estruturados para vulnerabilidades encontradas.

Destaques

  • Implementei analise sobre ASTs de Python para raciocinar sobre fontes, sinks e sanitizadores.
  • Suportei varios padroes de vulnerabilidade, incluindo XSS, SQL injection, SSRF e path traversal.
  • Tratei fluxos de informacao explicitos e implicitos com formatacao deterministica dos resultados.

Tecnologias

  • Python
  • Parsing de AST
  • Padroes de vulnerabilidade em JSON
  • Analise estatica de programas
